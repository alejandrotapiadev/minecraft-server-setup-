Servidor Dedicado Minecraft en Ubuntu Server

Requisitos Actualizar Sistema Instalar Java Crear Usuario Descargar Minecraft Configurar EULA Iniciar Servidor Consejos

Requisitos

  • Servidor con Ubuntu Server 18.04 o superior.
  • Acceso root o sudo.
  • Conexión a internet.
  • Conocimientos básicos de terminal.

Actualizar Sistema

Ejecuta los siguientes comandos para actualizar el sistema:

sudo apt update
sudo apt upgrade -y

Instalar Java

Minecraft Server requiere Java 17 o superior. Instala OpenJDK con: 

sudo apt install openjdk-17-jre-headless -y

Verifica la instalación:

java -version

Crear Usuario para el Servidor

Para seguridad, crea un usuario dedicado:

sudo adduser minecraft
sudo su - minecraft

Descargar Minecraft Server

Descarga la última versión del servidor Minecraft:

wget https://launcher.mojang.com/v1/objects/1b34c5a70d60cba5d3b2a7dbd8db8c6180d6b408/server.jar -O minecraft_server.jar

(Nota: reemplaza el URL con la última versión oficial.)

Aceptar el EULA

Ejecuta el servidor por primera vez para crear archivos:

java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui

Luego edita el archivo eula.txt para aceptar el EULA: 

nano eula.txt

Cambia eula=false a eula=true y guarda.

Iniciar el Servidor

Inicia el servidor con:

java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui

Para dejarlo corriendo en segundo plano, puedes usar screen o tmux: 

sudo apt install screen
screen -S minecraft
java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui
# Para desconectar sin cerrar el servidor, presiona Ctrl + A, luego D

Consejos Adicionales

  • Configura el firewall para permitir el puerto 25565 (TCP): sudo ufw allow 25565
  • Realiza backups periódicos del mundo.
  • Modifica la configuración del servidor en server.properties.
  • Considera usar servicios como systemd para gestionar el servidor.
